## Why go-pomodoro?

Most Pomodoro applications are GUI-based and distracting.
**go-pomodoro** is designed for developers who prefer:

* Terminal-first workflow
* Minimal and fast productivity tools
* Scriptable CLI utilities
* Zero unnecessary UI interruptions

---

## Features

* Terminal-based Pomodoro timer
* Default 25-minute focus session
* Custom duration support
* Lightweight single-binary distribution
* Cross-platform support (Windows / Linux / macOS)
* Simple and distraction-free interface

## Usage

After installation, you can use the Pomodoro CLI from anywhere in your terminal.

### Start a Pomodoro session (default 25 minutes)

```bash
pomodoro start
```

### Start a Pomodoro session with custom duration

```bash
pomodoro start 15
```

### Stop the current session

```bash
pomodoro stop
```

### Check session status

```bash
pomodoro status
```
